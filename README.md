# fivem_rizzler
Rizz up NPCs in FiveM

# Installation Instructions
Download as zip  
Unzip  
Drag files into resources  
Add `ensure dw_rizz` to your server.cfg  
Drag dw_rizz.ogg into `interact-sound/client/html/sounds`  
Lose all self respect
